# Images

Deze map bevat afbeeldingen en screenshots voor de Kubernetes cluster installatie handleiding met Rancher.

## Gebruik

Plaats hier screenshots, diagrammen en andere visuele hulpmiddelen die helpen bij het begrijpen van de installatiestappen.

### Aanbevolen bestandsformaten
- PNG voor screenshots
- SVG voor diagrammen en schema's
- JPG voor foto's

### Naamgeving
Gebruik duidelijke en beschrijvende bestandsnamen, bijvoorbeeld:
- `haproxy-configuratie.png`
- `kubernetes-cluster-overzicht.svg`
- `rancher-dashboard.png`
